# Fitness Tracker - QUICK START GUIDE

## Easiest Way to Get Started (3 Simple Steps!)

### Step 1: Extract the Files
- Right-click the ZIP file
- Click "Extract All"
- Choose where to save it
- Open the extracted folder

### Step 2: Start the App
- **DOUBLE-CLICK** the file called `START-SERVER.bat`
- A black window will open - **KEEP IT OPEN**
- Your web browser should automatically open

### Step 3: Use the App
- Click "Launch Fitness Tracker" button
- Start tracking your workouts and meals!

**That's it! You're ready to go!**

---

## What You Can Do

### Track Workouts
- Add exercises with sets, reps, and weight
- Check them off as you complete them
- Track different days with the date picker

### Track Nutrition
- Search from 40+ pre-loaded foods
- See calories, protein, carbs, and fat
- Adjust portion sizes
- Add to breakfast, lunch, or dinner
- View daily totals

### View Your Progress
- See total exercises completed
- Check daily calories and macros
- Switch between different dates

## Quick Tips

1. **Keep the black window open** while using the app (it's the server)
2. **To close**: Just press `Ctrl+C` in the black window or close it
3. **To open again**: Double-click `START-SERVER.bat` anytime
4. **Your data is saved** automatically in your browser
5. **Use the date picker** at the top to see different days

## Using on Your Phone

### Same WiFi Method (Quick & Easy)
1. Make sure your phone and computer are on the same WiFi
2. Start the app on your computer (double-click START-SERVER.bat)
3. Look at the black window - it shows a URL like `http://192.168.X.X:8000`
4. Type that URL into your phone's browser
5. Bookmark it for easy access!

### Install as Phone App
See `INSTALLATION-GUIDE.md` for detailed instructions on installing this as an actual app on your iPhone or Android.

## Food Search Tips

The app includes 40 common foods:
- **Proteins**: Chicken, Salmon, Tuna, Beef, Turkey, Eggs, etc.
- **Grains**: Rice, Oatmeal, Quinoa, Pasta, Bread
- **Veggies**: Broccoli, Spinach, Sweet Potato, Carrots
- **Fruits**: Banana, Apple, Orange, Berries, Avocado
- **Dairy**: Greek Yogurt, Milk, Cheese
- **Nuts**: Almonds, Walnuts, Peanut Butter
- **Others**: Pizza, Burger, Fries, Ice Cream

To search: Just type 2+ letters (like "chi" for chicken)

## Need Help?

**App won't start?**
- Make sure Python is installed on your computer
- The app needs Python to run the local server

**Data disappeared?**
- Don't clear your browser data/cookies
- Each browser saves data separately
- Don't use private/incognito mode

**Want to use on phone permanently?**
- See `INSTALLATION-GUIDE.md` for hosting options
- Or just use the same WiFi method whenever needed
