How to Install on iPhone:

  1. Open Safari on your iPhone (must use Safari, not Chrome or other browsers)
  2. Navigate to the fitness-tracker.html file
  3. Tap the Share button (square with arrow pointing up) at the bottom of the screen
  4. Scroll down and tap "Add to Home Screen"
  5. Edit the name if desired, then tap "Add"
  6. The app icon will appear on your home screen like a native app

How to Install on Android:

  1. Open Chrome on your Android device
  2. Navigate to the fitness-tracker.html file
  3. Tap the menu (three dots) in the top right
  4. Tap "Add to Home screen" or "Install app"
  5. Confirm the installation
  6. The app will appear on your home screen

Setup Instructions:

  1. Download both icons using the buttons above
  2. Save them in the same folder as your fitness-tracker.html file
  3. Make sure all files are in the same directory:
    • fitness-tracker.html
    • manifest.json
    • sw.js
    • icon-192.png
    • icon-512.png
  4. For iPhone installation, you'll need to serve the files via a web server (or upload to a hosting service)
